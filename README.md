Анализ пользовательского поведения и эффективности маркетинговых каналов

В рамках данного проекта был проведён анализ пользовательского поведения на сайте сервиса по подбору автомобилей с целью оценки эффективности каналов привлечения и выявления факторов, влияющих на конверсию.

Проект выполнен в рамках дипломной работы на платформе Skillbox.

Цели исследования:
- Оценить качество трафика из различных источников;
- Проанализировать влияние устройства пользователя на конверсию;
- Проверить влияние географии пользователей на поведение;
- Определить наиболее эффективные каналы привлечения;
- Выявить автомобили с наибольшим спросом и конверсией.

В проекте использованы данные веб-аналитики:
- sessions (информация о сессиях пользователей);
- hits (действия пользователей на сайте).

Ключевые признаки:
- источник трафика (utm_source, utm_medium);
- устройство пользователя (device_category);
- география (geo_city);
- события (просмотры, действия).

Используемые инструменты:
- Python (pandas, numpy);
- Визуализация (matplotlib, seaborn).

Основные этапы работы:
1. Предобработка данных: очистка пропусков, приведение типов данных, удаление нерелевантных признаков.
2. Разведочный анализ (EDA): анализ распределений, изучение структуры трафика, выявление аномалий.
3. Проверка гипотез.

Были проверены следующие гипотезы:
1. Конверсия органического и платного трафика не отличается.
2. Конверсия мобильных и десктопных пользователей не отличается.
3. Конверсия пользователей из разных регионов не отличается.
Для проверки использовался z-тест пропорций.

Ключевые результаты:
- Платный трафик демонстрирует более высокий уровень конверсии по сравнению с органическим;
- Конверсия пользователей зависит от типа устройства;
- Географический фактор не оказывает значимого влияния на конверсию.
Бизнес-инсайты:
- Наиболее популярные автомобили не всегда имеют наивысшую конверсию;
- Существуют различия между объёмом трафика и его качеством;
- Некоторые каналы показывают высокий CR, но имеют низкий объём.
  
Визуализация
В проекте использованы следующие типы графиков:
- столбчатые диаграммы (сравнение CR);
- линейные графики (динамика показателей);
- scatter plot (анализ объёма vs конверсии).

Ограничения:
- анализ проводился на подвыборке данных;
- возможны искажения из-за неполной атрибуции каналов ((none), (not set)).

Результаты анализа позволяют:
- оптимизировать распределение маркетингового бюджета;
- повысить эффективность каналов привлечения;
- улучшить пользовательский опыт.
